Computes the Jacobian for the time derivative of the signed distance functions vs. impulses applied at contact points.

Assume the signed distance function is Phi(q(t)), so Phi(q(t_0 + t))_{t_0} Phi(q(t_0)) + d/dt Phi(q(t_0)) * dt ==> d/dt Phi(q(t_0)) (Phi(q(t_0 + t)) - Phi(q(t_0))/dt
