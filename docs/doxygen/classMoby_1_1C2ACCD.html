Detailed Description

Implements the ContactFinder abstract class to perform exact contact finding using vertices against triangles.

C2ACCD::C2ACCD ( )

Constructs a collision detector with default tolerances.

eps_tolerance is set to NEAR_ZERO and TOI tolerance is set to 1e-2.
